# OpenMD

**OpenMD** is an open-source project written in C++ that focuses on implementing basic mathematical algorithms and applications.  
It is designed for learning, experimentation, and extending math-related functionality.

## Features
- Provides common mathematical algorithm implementations  
- Modular structure for easy extension  
- Includes test scripts for quick verification  

## Quick Start
```bash
git clone https://github.com/Travis-Lee/OpenMD.git
cd OpenMD
./run_test.sh
```

## Tech Stack
- C++  
- CMake  
- Shell
